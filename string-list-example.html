String List Plugin for Tweakpane

This plugin allows you to create and manage dynamic lists of strings using Tweakpane. Each string in the list uses TextController for consistent text input behavior, and you can add/remove items dynamically.

Note: This is a demonstration page. To actually use the plugin, you need to build the project and import the generated plugin bundle into your Tweakpane setup.

Usage Example

Here's how you would use the String List plugin in your project:

import {Pane} from 'tweakpane'; import * as TablePanePlugin from './path/to/dist/tweakpane-plugin-template.js'; // Register the plugin const pane = new Pane(); pane.registerPlugin(TablePanePlugin); // Create parameters object with string list const PARAMS = { stringList: ['Hello', 'World', 'Example', 'Strings'] }; // Add the string list input const stringListInput = pane.addInput(PARAMS, 'stringList', { view: 'stringlist', label: 'Items' }); // Listen for changes stringListInput.on('change', (ev) => { console.log('String list changed:', ev.value); // ev.value will be an array of strings });

Plugin Features

  • Dynamic Management: Add and remove string items with buttons
  • TextController Integration: Each string uses Tweakpane's native text input
  • Real-time Updates: Changes are immediately reflected in the bound object
  • Consistent Styling: Matches the existing Tweakpane theme
  • Type Safety: Full TypeScript support with proper type definitions

Plugin Parameters

interface StringListPluginInputParams { view: 'stringlist'; // Required identifier label?: string; // Label for string items (default: 'String') }

Build Instructions

To use this plugin in your project:

  1. Run npm run build to build the plugin
  2. Import the generated bundle from dist/tweakpane-plugin-template.js
  3. Register the plugin with your Tweakpane instance
  4. Use view: 'stringlist' when adding inputs

Technical Implementation

The plugin is built using the value pair list as a foundation but adapted for strings:

  • Model: StringList class managing an array of strings
  • Controller: Uses TextController for each string item instead of NumberTextController
  • View: Simplified single-input rows instead of dual number inputs
  • Styling: Consistent with existing Tweakpane design patterns
